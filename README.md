# Uniswap v3 Multi-Pool Token Analysis

Built a Uniswap v3 dashboard analyzing how a single token trades across multiple pools on Ethereum, with a focus on execution quality, flow, and market structure.

## Overview
This project aggregates Uniswap v3 swap data for a single ERC-20 token across all available pools on Ethereum.
It focuses on understanding pricing behavior, volume distribution, and cross-pool structure.

## Dashboard
- Platform: Dune Analytics  
- Scope: Uniswap v3 (Ethereum)
- Token case study: REKT

## Metrics Included
- Swap count (per pool & global)
- Token flow (pool & global)
- Volume-weighted average price (VWAP)
- Highest observed price per pool
- Tick-based price behavior

## Notes & Limitations
- Liquidity metrics were excluded due to ambiguity in Uniswap v3 active liquidity interpretation.
- Analysis focuses on executed swaps only.
